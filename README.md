## 使用说明

1. **准备工作**:
   - 确保您的九联UNT401H机顶盒已连接到网络。
      - 准备一个格式化后的U盘，用于存放刷机包。

      2. **刷机步骤**:
         - 将刷机包文件复制到U盘的根目录。
            - 将U盘插入机顶盒的USB接口。
               - 进入机顶盒的ADB调试模式，选择U盘中的刷机包进行安装。
                  - 等待刷机过程完成，机顶盒将自动重启并进入当贝桌面纯净版系统。

                  3. **注意事项**:
                     - 刷机过程中请勿断电或拔出U盘，以免造成系统损坏。
                        - 刷机前请备份重要数据，以防数据丢失。
